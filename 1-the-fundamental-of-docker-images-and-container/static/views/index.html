Build a Docker image

  • 1. Build an image.

    $ docker build .

  • 2. Run detached container with port mapping.

    $ docker run -p PORT:PORT -d IMAGE

  • 3. Entering interactive mode to a running container.

    $ docker exec -it CONTAINER sh

  • 4. Print logs of the detached container.

    $ docker logs [OPTIONS] CONTAINER

  • 5. Inspect an image.

    $ docker inspect IMAGE

  • 6. Copy files into or from a container.

    Copy files into a container (not recommended).

    $ docker cp [OPTIONS] SRC CONTAINER:DEST

    Copy files from a container.
    Use cases: Extract log files from the container.

    $ docker cp [OPTIONS] CONTAINER:DEST SRC

  • 7. Tag images.

    $ docker build -t IMAGE_NAME:TAG .

  • 8. Name containers.

    $ docker run -p PORT:PORT -d --rm --name CONTAINER_NAME IMAGE
